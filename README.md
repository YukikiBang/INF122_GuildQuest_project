# GuildQuest (Python version)

## Overview
This is a CLI implementation of the GuildQuest application base on the rank 1 UML design. The program allows users to manage campaigns, quest events, realms, characters, inventories, timelines, and sharing permissions.

# Requirements
- Python 3.13
- No third-party libraries are required

# How to run the program
1. Unzip the submission folder
2. Open the program directory
3. Run the program with main.py
4. After run the program check the menu option and start the program
